Level Structure †

Appointments in CC = ∫Booking Rate+Rescheduling Rate-Completing Rate dt + [Initial Appointments]
Initial Appointments = Appt Completion Rate*RVI used when Booking
    Booking Rate = (Start Rate/Patients per Appointment)+(ZIDZ(ZIDZ(Patients in CC , Patients per Appointment) , RVI used when Booking))
    Completing Rate = Appointments for Existing Patients
    Rescheduling Rate = Appointments to Reschedule/Time to Reschedule

Appointments to Reschedule = ∫True Missed Appointment Rate-Rescheduling Rate dt + [True Missed Appointment Rate*Time to Reschedule]
Time to Reschedule = 2
True Missed Appointment Rate = MAX(Completing Rate*Effect of Task Time on Missed Appointments*"Actual Missed Appointment %",0)

Patients in CC = ∫Start Rate-Ending Rate dt + [Initial Patients]
Initial Patients = Patient Start Rate based on Supply and RVI and Missed Appt pct and Engagement Duration*Engagement Duration
    Ending Rate = ZIDZ(Patients in CC,Engagement Duration)
    Start Rate = MIN(Maximum Start Rate,Appointments Available for New Patients*Patients per Appointment)

Waiting to Start = ∫Referral Rate-Start Rate dt + [Initial Patients Waiting to Start]
Initial Patients Waiting to Start = Patient Start Rate based on Supply and RVI and Missed Appt pct and Engagement Duration*Target New Patient Wait Time
    Referral Rate = IF THEN ELSE(User Referral Rate>0:AND:Time>Experiment Week, User Referral Rate, Referral Rate inferred from Data)*Effect of Wait Time on Referrals
